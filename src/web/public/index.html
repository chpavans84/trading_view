✅ Trade Entry Checklist

Steps the bot runs on every scan (in order):
  1. Daily P&L → stop if target hit or loss limit reached
  2. Market status → skip if closed
  3. Positions → stop if at max open positions
  4. VIX → stop if above max-VIX setting
  5. Build candidate list (movers + earnings + sectors)
  6. Filter to 4–6 best candidates
  7. Run conviction score on each → skip if below threshold
  8. Execute highest-scoring candidate

🎯 Conviction Score Factors (0–100, fixed weights)

[TV] factors only apply when TradingView Desktop is connected on port 9222. A backtest-weighted adjustment (±10 pts) is applied automatically based on historical alpha of the final grade.
